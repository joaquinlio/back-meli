# Arquitectura Back-meli

## Stack
* Node.js / Typescript
* Express
* Nodemon

## Para iniciar la aplicación en modo desarrollo

1 - Instalar depedencias
```
$ npm install
```

3 - Levantar entorno
```
$ npm run dev
```

## Para Implementar la aplicación en modo productivo

1 - Instalar depedencias
```
$ npm install
```

2 - Buildear solución
```
$ npm run compile
```

3 - Levantar entorno 
```
$ npm run server
```
